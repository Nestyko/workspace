# Raw Sources

Drop immutable source documents here: clipped articles, exported PDFs,
Confluence exports, meeting transcripts, interview notes, spec dumps.

- This directory is **gitignored** — sources are local-only and never committed.
- Put referenced images/attachments in `assets/`.
- The agent reads from here but never modifies these files.
- See `../SCHEMA.md` for the full ingest workflow.
