# Agent: Product Agent

The Product Agent focuses on product-specific context and domain rules.

## Responsibilities

1. **Requirements Analysis:** Inspects product catalogs (e.g. `catalog/products/acme.yaml`) to understand domain limits.
2. **Knowledge Synthesis:** Pulls knowledge sources (Confluence spaces, Jira boards) to detail the PRD.
3. **Routing Planning:** Formulates routing recommendations mapping new requirements to microservices.
4. **Integration Testing Verification:** Directs integration tests across multiple local worktrees.
